# Einrichtung

Um in diesem Kurs die praktischen Aufgaben zu lösen, ist es nötig die Arbeitsumgebung einzurichten. Hierzu beöntigen Sie Programme auf dem System mit welchen Sie arbeiten werden. Da die Einrichtung während des ersten Praktikums durchgeführt wird, dient dies hier lediglich als Kurzanleitung.

Virtual Studio Code (VSC)

VSC ist ein Texteditor mit welchem die enstsprechenden Codezeilen geschrieben werden können. VSC ist im Bereich der Webentwicklung der Industriestandard. Durch seine Schlankheit ist VSC nicht Performancelastig und gleichzeitig kann es durch Erweiterungen so angepasst werden, dass es für alle Bereiche der Entwicklung passend ist

Installation

Sie finden VSC hier.
Einen mit einem Klick auf den Pfeil (siehe Abbildung) können Sie die MacOs (Apple-Mac) version herunterladen. Achten Sie daran, immer den Stable-Build zu nehmen.
